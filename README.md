# YouTube Audio Loader

Desktop app to download audio from YouTube and drag it to your DAW.

## Features
- Download MP3 from YouTube
- 320kbps quality
- Drag & drop to DAW (FL Studio, Ableton, etc.)

## Requirements
- Python 3.x
- PyQt5
- yt-dlp
- ffmpeg

## Install
```bash
pip install PyQt5
pip install yt-dlp
```

## Usage
```bash
python YouTubeLoader.py
```

1. Paste YouTube URL
2. Click DOWNLOAD MP3
3. Drag file from folder to your DAW
